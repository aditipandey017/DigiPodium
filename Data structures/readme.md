# data structures in python

## list
        - mutable
        - indexed
        - ordered
        - can contain duplicates
## tuple
        - immutable
        - indexed
        - ordered
        - can contain duplicates
## set
        - mutable
        - unindexed
        - unordered
        - can contain only unique values
    
## dict
        - mutable
        - unindexed
        - ordered
        - can contain only unique keys
        - can contain duplicates values
